# AIOX Onboarding — Master Path to Framework Expertise

Learn Synkra AIOX from foundation to mastery through structured phases, interactive scenarios, and decision-making exercises.

## Quick Start

```
/aiox-onboarding                    # Interactive guided path (default)
/aiox-onboarding phase-1            # Foundation only
/aiox-onboarding phase-2            # Mental Maps only
/aiox-onboarding phase-3            # Processing Maps only
/aiox-onboarding phase-4            # Mastery only
/aiox-onboarding all                # All 4 phases sequentially
/aiox-onboarding all --quiz         # With knowledge quizzes
/aiox-onboarding all --scenario     # With real-world scenarios
```

## The 4 Learning Phases

### Phase 1: Foundation (10 minutes)
**Goal:** Understand what AIOX is and who the agents are.

**Topics:**
- What is Synkra AIOX? (AI-Orchestrated System for Full Stack Development)
- The 13 agents and their roles
- Core concepts: Constitution, Story-Driven Development, Authority
- Framework boundaries (L1-L4)
- Key terminology

**Learning Path:**
1. Overview of AIOX (2 min)
2. Meet the 13 agents (4 min)
3. Core principles intro (2 min)
4. Framework structure (2 min)

**Deliverable:** Understanding of framework basics and agent landscape.

---

### Phase 2: Mental Maps (15 minutes)
**Goal:** Internalize AIOX DNA — core patterns, philosophy, mental models.

**Topics:**
- 8 Core DNA patterns (Agent Specialization, Story-Driven, Quality Gates, etc.)
- Constitution articles (6 inviolable principles)
- Authority hierarchy and delegation patterns
- Framework philosophy: "Freedom on rigid foundation"
- Thinking patterns for decision-making

**Learning Path:**
1. DNA pattern deep-dive (5 min)
2. Constitution explained (4 min)
3. Authority hierarchy visualization (3 min)
4. Mental model synthesis (3 min)

**Deliverable:** Internalized framework philosophy and decision-making patterns.

---

### Phase 3: Processing Maps (20 minutes)
**Goal:** Master the workflows and execution pipelines.

**Topics:**
- Story Development Cycle (SDC): 5 phases with gates
- Spec Pipeline: 6-phase requirements transformation
- QA Loop: Iterative review-fix cycle
- Brownfield Discovery: 10-phase technical debt assessment
- Quality gates: 7 mandatory checks
- Decision trees for workflow selection

**Learning Path:**
1. SDC deep-dive (7 min)
2. Other workflows explained (5 min)
3. Quality gates & gates (4 min)
4. Workflow decision tree (4 min)

**Deliverable:** Ability to navigate and execute workflows correctly.

---

### Phase 4: Mastery (25 minutes)
**Goal:** Real-world decision-making and advanced execution patterns.

**Topics:**
- Agent delegation scenarios
- Authority boundary enforcement
- Handling boundary violations
- Quick Flow vs Standard vs Enterprise workflows
- Real-world problem-solving
- Advanced patterns and anti-patterns
- Escalation procedures

**Learning Path:**
1. Scenario 1: Story creation flow (5 min)
2. Scenario 2: Authority question (5 min)
3. Scenario 3: Workflow selection (5 min)
4. Scenario 4: Boundary violation (5 min)
5. Self-assessment & next steps (5 min)

**Deliverable:** Confidence to navigate real AIOX scenarios and make authority decisions.
