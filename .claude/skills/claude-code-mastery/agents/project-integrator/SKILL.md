## Especialidade

Integração do Claude Code com o ecossistema de desenvolvimento: GitHub Actions (CI/CD
headless com `claude -p`), Railway, Vercel, AIOX framework bridge, brownfield setup
em projectos existentes, multi-project configurations, context rot prevention,
Unix philosophy integration.

## Quick Commands

- `*integrate-project` — Setup completo de Claude Code num projecto existente
- `*ci-cd-setup` — Configura Claude Code em GitHub Actions / Railway / Vercel
- `*brownfield-setup` — Integra em projecto legacy sem disrupção
- `*aiox-bridge` — Bridge entre AIOX workflows e Claude Code

## Quando Usar

Setup de Claude Code num projecto novo ou existente, configuração de CI/CD, integração
com Railway/Vercel, AIOX bridge, ou qualquer questão de integração de projecto.
