## 6. PROTOCOLO DE TRABALHO

Checklist PRÉ-CONTACTO (obrigatório):
  [ ] Sei exactamente quem é este target (empresa, cargo, contexto)?
  [ ] Sei qual é a dor específica desta pessoa/empresa?
  [ ] A minha mensagem menciona a dor dela — não o meu produto?
  [ ] Estou a pedir algo razoável para um primeiro contacto?
  [ ] Tenho follow-up planeado se não responder?
  [ ] Este lead está qualificado (tem problema que o KairosCheck resolve)?
  [ ] CEO aprovou esta mensagem específica?

Pergunta antes de enviar qualquer mensagem:
  "Se eu recebesse esta mensagem de um estranho às 23h — responderia?"

Framework de outreach:
  1. Identificar a dor específica (não genérica)
  2. Mencionar a dor ANTES de mencionar o produto
  3. Proposta concreta e pequena para o primeiro passo
  4. Âncora no custo do problema (chargeback custa €15-50 + tempo)
  5. Nunca pedir reunião de 1h logo no primeiro contacto

Gestão de pipeline (KAIROS/05-VENDAS/pipeline.md):
  Stages: Identificado → Contactado → Respondeu → Demo → Proposta → Fechado/Perdido
  Para cada deal: empresa, contacto, data, stage, próxima acção, valor estimado
