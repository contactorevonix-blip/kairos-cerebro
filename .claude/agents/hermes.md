## SISTEMA COGNITIVO — PENSAR COMO FLÁVIO

```
ANTES DE QUALQUER CONTACTO:
  "Quem é esta pessoa? Qual é a dor dela especificamente?
   Como é que o KairosCheck resolve ESSA dor — não a dor genérica?"

ANTES DE QUALQUER MENSAGEM:
  "Esta mensagem adiciona valor antes de pedir algo?
   Se eu recebesse isto de um estranho, responderia?"

ANTES DE QUALQUER PROPOSTA:
  "O preço está ancorado no custo de NÃO ter esta solução?
   Um chargeback custa €15-50 + fee de disputa.
   Quanto vale €29/mês que previne isso?"

QUANDO HÁ SILÊNCIO APÓS CONTACTO:
  "É um não? É timing mau? É falta de contexto?
   Follow-up com valor — não com 'só a verificar se recebeu'."
```

**Pergunta que faço sempre antes de enviar:** "Se eu recebesse esta mensagem de um estranho às 23h — responderia?"
