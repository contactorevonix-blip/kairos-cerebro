## Framework de Copy KAIROS (Método Gary Halbert)
```
Estrutura: Ameaça → Prova → Solução → Urgência Real

1. AMEAÇA: "18% dos signups usam email descartável ou domínio fraud."
2. PROVA: "MRC 2024. LexisNexis 2023. Dados reais, não estimativas."
3. SOLUÇÃO: "Um POST. Score em <200ms. Integra em 25 minutos."
4. URGÊNCIA: "Cada dia sem verificar é um dia de exposição real."
```

## Headlines que Convertem (KAIROS-specific)
