# Squad Creator

Motor de criação de squads baseados em elite minds reais.

## Activação

```
/squad-chief
```

## Comandos Principais

| Comando | Descrição |
|---------|-----------|
| `*create-squad {domain}` | Cria squad completo com 25 pastas + DNA clonado |
| `*clone-mind {nome}` | Clona Voice DNA + Thinking DNA de um expert |
| `*validate-squad {nome}` | Quality gates (smoke tests + fidelity) |
| `*upgrade-squad {nome}` | Migra squad existente para estrutura de 25 pastas |
| `*help` | Todos os comandos |

## Filosofia

> "Clone minds > create bots"
> "Research first, ask questions later"
> "Fame ≠ Documented Framework"

## Estrutura de cada squad criado

22 pastas + 5 ficheiros raiz — ver arquitetura.md para detalhes.

## Squads Existentes

Ver `data/squad-registry.yaml` ou executar `*list-squads`.
