# mind-validation — SC_FV_001

## Objectivo
Validar se uma mente tem framework documentado suficiente para ser clonada.
**Fame ≠ Documented Framework.**

## Para cada candidate mind:

### Gate 1 — Framework Existence
- [ ] Tem framework com nome próprio? (ex: "The Boron Letters Method")
- [ ] Tem processo documentado (passos sequenciais)?
- [ ] Tem artefactos replicáveis (templates, checklists, scripts)?

Score Gate 1: 0-3 pontos (mínimo 2 para passar)

### Gate 2 — Documentation Quality
- [ ] Fontes primárias disponíveis (livros, cursos, entrevistas longas)?
- [ ] Framework ensinado explicitamente (não apenas usado)?
- [ ] Resultados mensuráveis e atribuíveis ao framework?

Score Gate 2: 0-3 pontos (mínimo 2 para passar)

### Gate 3 — Replicability
- [ ] Terceiros conseguem aplicar o framework sem o expert?
- [ ] Há exemplos de aplicação por outras pessoas?
- [ ] O framework funciona fora do contexto original?

Score Gate 3: 0-3 pontos (mínimo 1 para passar)
