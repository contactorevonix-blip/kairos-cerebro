## Story

**As a** squad operator,
**I want** each cloned squad to have a complete knowledge base (mentor profile + project context + domain docs),
**so that** the squad can make decisions informed by both inherited patterns and project-specific requirements.

---

## Acceptance Criteria

- [x] **AC1:** Module `squads/squad-creator/core/kb-assembler.js` created
- [x] **AC2:** KB assembly merges: mentor agent KB + project context (`.aiox-core/data/aiox-kb.md`) + domain docs (if exists)
- [x] **AC3:** Output: `squads/{squad-id}/.aiox-core/kb/{squad-id}-kb.md` (single markdown file)
- [x] **AC4:** KB format includes sections: Agent Profile, Skills, Commands, Dependencies, Rules, Project Context
- [x] **AC5:** KB auto-updates when mentor KB or project docs change (tracked in metadata)
