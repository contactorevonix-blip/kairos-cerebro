### Design Decisions
- **Format:** JSON (not YAML, not JSONL) — no external dependencies, aligned with engine.js
- **Atomicity:** Temp file + rename for concurrent write safety
- **API:** write(sessionId, contextState), query(intentType), delete, getAll, read, clear
- **Validation:** Enforce required fields (timestamp, intent_type, completeness, phase_4_passed)
