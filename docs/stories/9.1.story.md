## Description

Implement base infrastructure for Constitutional enforcement gates at the PreToolUse hook layer. All 7 Constitutional articles (Art. I-VII) will hook into this framework to enforce policy deterministically.

## Acceptance Criteria

- [ ] Create `.claude/hooks/gate-framework.cjs` with gate-runner interface
- [ ] Gate-runner loads gate modules (enforce-agent-authority, enforce-story-driven, etc)
- [ ] Each gate decision logged to `.aiox/gate-logs/` as immutable JSONL (timestamp, article, decision, reason)
- [ ] Decision verdicts: `allow`, `block`, `override`, `warn`
- [ ] No gate can crash the hook (graceful error handling)
- [ ] Metrics: `gatesEnforced`, `violationsDetected`, `violationsBlocked`, `overridesUsed` tracked
- [ ] All 4 pre-commit gates PASS (lint ✅, typecheck ✅, test ✅, CodeRabbit pending)
