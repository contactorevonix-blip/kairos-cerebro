## Contexto Técnico

Trabalho realizado fora do ciclo formal de stories (spike / entrega directa). Dois ficheiros criados na raiz do projecto `KAIROS_CEREBRO`:

**`kairos-server.js`** — Servidor HTTP local (Node.js) que:
- Serve o dashboard no browser (localhost)
- Expõe endpoint de dados em tempo real
- Lê e agrega informação do projecto (stories, git, métricas)

**`kairos-snapshot.js`** — Script de snapshot que:
- Recolhe estado actual do projecto KAIROS_CEREBRO

**Output gerado:**
- `C:/Users/lealp/OneDrive/Downloads/kairos-data.json` — snapshot do estado do projecto
