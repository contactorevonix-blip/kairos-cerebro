## Description

Implement Constitutional Article II (Agent Authority) enforcement gate. Block @devops-exclusive operations (git push, PR create, release) when invoked by non-@devops agents. Gate hooks at PreToolUse layer.

**Article II Statement:** @devops has EXCLUSIVE authority over git push, gh pr create, and release operations. No other agent may perform these.

## Acceptance Criteria

- [ ] Gate reads active agent from hook context (env vars → inline var → session)
- [ ] Detect @devops-exclusive operations: `git push`, `gh pr create`, `gh pr merge`, `release`
- [ ] Block if agent ≠ @devops (decision: `block`)
- [ ] Allow `--skip-devops-check` override (rare, logged as `override`)
- [ ] Decision logged: article=`art-ii-agent-authority`, reason explains violation
- [ ] Test cases: 
  - [ ] @dev attempts `git push` → BLOCKED
  - [ ] @devops attempts `git push` → ALLOWED
  - [ ] @sm attempts `gh pr create` → BLOCKED
  - [ ] @devops + `--skip-devops-check` → OVERRIDE logged
- [ ] Zero violations in 2-week gate-logs (baseline)
- [ ] All 4 pre-commit gates PASS
