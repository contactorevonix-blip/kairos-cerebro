## Story

**As a** developer a usar o KAIROS_CEREBRO,
**I want** poder activar qualquer agente dos 4 squads via slash command (`/Deep-Research:agents:booth`, `/System-Factory:agents:forge-classifier`, etc.),
**so that** todos os squads funcionem igual ao Claude-Code-Mastery que já está sincronizado.

---

## Contexto Técnico

Padrão AIOX: cada squad activo no `.aiox-sync.yaml` deve ter:
1. `.claude/commands/{Alias}/agents/{agent}.md` — shim de activação via slash command
2. `.claude/skills/{squad}/agents/{agent}/SKILL.md` — skill canonical de activação

Existe para: `AIOX/`, `Chiefs/`, `Claude-Code-Mastery/`
Falta para: `Deep-Research/`, `Squad-Creator/`, `AIOX-Cerebro/`, `System-Factory/`

**Contagem de ficheiros a criar:**
- Deep-Research: 11 agentes × 2 = 22 ficheiros
- Squad-Creator: 3 agentes × 2 = 6 ficheiros
- AIOX-Cerebro: 1 agente × 2 = 2 ficheiros
- System-Factory: 6 agentes × 2 = 12 ficheiros
- **Total: 42 ficheiros**

---

## Acceptance Criteria

1. `/Deep-Research:agents:booth` activa o agente booth correctamente
2. `/System-Factory:agents:forge-classifier` activa forge-classifier correctamente
3. `/Squad-Creator:agents:oalanicolas` activa oalanicolas correctamente
4. `/AIOX-Cerebro:agents:aiox-cerebro` activa aiox-cerebro correctamente
5. Formato dos shims é idêntico ao padrão `Claude-Code-Mastery`
6. Formato dos SKILL.md é idêntico ao padrão `claude-code-mastery`
7. Todos os 42 ficheiros criados nos paths correctos
