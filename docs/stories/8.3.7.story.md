**Status:** Done

## Story

**As a** squad customizer,
**I want** cloned squads to inherit mentor rules while allowing squad-specific overrides,
**so that** domain-specific teams can follow framework rules but adapt them for their context.

---

## Acceptance Criteria

- [x] **AC1:** Rule inheritance system copies mentor's `.claude/rules/` to squad context
- [x] **AC2:** Squad can override specific rules via `squads/{squad-id}/.claude/rules/squad-overrides.md`
- [x] **AC3:** Validation: @qa gate confirms no rule conflicts with framework constitution (Articles I-VII)
- [x] **AC4:** Documentation file `squads/{squad-id}/.claude/rules/squad-overrides.md` lists all inherited rules + documented overrides
- [x] **AC5:** Example override: Squad for UI work might override `absolute-imports.md` for CSS modules
