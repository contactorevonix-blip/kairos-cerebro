# Story 9.3: Art. III Story-Driven Development Enforcement Gate

## Description

Implement Constitutional Article III (Story-Driven Development) enforcement gate. Block git commits that don't reference an active story + ACs, or use `[no-story-req]` override for config-only commits.

**Article III Statement:** No code is written without a story associated. Every commit must trace to a story ID (e.g., `[Story 13.2]`) or use `[no-story-req]` override for exceptions.

## Scope

**IN:**
- Parse commit message for story reference regex
- Detect git commit operations at PreToolUse hook
- Block commits without story or override
- Log overrides with `[no-story-req]` flag
- Graceful degradation if story files unreadable
- 3+ test cases

**OUT:**
- Story creation or editing (that's @sm/@po)
- Commit message rewriting
- Story content validation (just story ID presence)
- Backfill existing commits

## Acceptance Criteria

- [ ] Gate detects `git commit` at PreToolUse hook
- [ ] Parse commit message for story reference: regex `\[Story \d+\.\d+\]`
- [ ] If no story reference AND no `[no-story-req]` flag → BLOCK
- [ ] If `[no-story-req]` present → ALLOW but log as `override`
- [ ] Decision logged: article=`art-iii-story-driven`, reason shows commit msg + decision
- [ ] Test cases:
  - [ ] `git commit -m "feat: add X [Story 13.2]"` → ALLOWED
  - [ ] `git commit -m "fix: typo"` → BLOCKED (no story)
  - [ ] `git commit -m "chore: update config [no-story-req]"` → OVERRIDE logged
- [ ] Graceful degradation: if story file unreadable, downgrade to `warn-and-proceed`
- [ ] All 4 pre-commit gates PASS

## Risks Identified

| Risk | Likelihood | Impact | Mitigation |
|------|-----------|--------|-----------|
| Regex false positive/negative | Medium | Medium | Test regex with 10+ commit message formats |
| Story files unreadable during validation | Low | Medium | Graceful downgrade to warn-and-proceed |
| Override flag abuse (config commits become excuse) | Low | Low | Audit trail; periodic review of [no-story-req] usage |
| Multi-story commits (3+ stories in one commit) | Low | Low | Require ONE story per commit (enforce single story) |
