## Description

Implement core SYNAPSE layer loader engine that orchestrates deterministic loading of all 8 context layers on agent activation. This is the foundation for all subsequent layer-specific loaders (13.2-13.6).

## Acceptance Criteria

- [ ] Layer loader validates 8-layer DAG (directed acyclic graph) before execution
- [ ] Loads Constitution (Art. I-VII) from `.aiox-core/constitution.md`
- [ ] Loads Global rules from `.claude/rules/agent-authority.md`, `workflow-execution.md`, `ids-principles.md`
- [ ] Implements atomic transitions (all-or-nothing, no partial loads)
- [ ] Logs all layer load events to `.aiox/context-load-logs/{date}.jsonl`
- [ ] Validates 100% coverage (no missing layers)
- [ ] Performance: cold start <2s, cached <500ms
- [ ] All 4 pre-commit gates PASS (lint, typecheck, test, CodeRabbit)

## Technical Notes

**Key Technical Decisions to Validate:**
1. Layer load sequence: Constitution → Global → Agent → Workflow → Task → Squad → Keyword (deterministic order)
2. Atomic transactions: Rollback on any layer failure
3. Cache format: JSON snapshots in `.synapse/` or in-memory LRU?
4. Error handling: Fail-fast or retry?
