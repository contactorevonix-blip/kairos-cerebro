## Story
**As a** developer do Kairos Check,
**I want** ter um ficheiro único `styles.js` com todos os tokens CSS e estilos base partilhados,
**so that** qualquer mudança de design se faça num único lugar e propague para todas as páginas.

## Acceptance Criteria

1. Ficheiro `packages/sniper-api/styles.js` criado com todos os tokens CSS e estilos base
2. Os 12 tokens de cor (`--bg`, `--surface`, `--accent`, etc.) definidos num único lugar
3. Tipografia (Inter + JetBrains Mono via Bunny Fonts) declarada num único lugar
4. Todas as 9 páginas importam `styles.js` em vez de definir CSS próprio
5. Zero CSS duplicado entre ficheiros de página para tokens e estilos base (verificar com grep)
6. Todas as 9 páginas renderizam visualmente igual ao estado anterior (sem regressões)
7. `packages/web/` removido (só continha `node_modules` — zero funcionalidade)

**Tokens de cor a centralizar** (fonte: `trust-pages.js`):
```css
:root {
  --bg: #0a0a0a;
  --surface: #111111;
  --surface-2: #1a1a1a;
  --border: #1f1f1f;
  --border-strong: #2a2a2a;
  --text: #f5f5f5;
  --text-secondary: #a3a3a3;
  --text-tertiary: #737373;
  --accent: #00d97e;
  --accent-hover: #00b369;
  --danger: #ef4444;
  --warning: #f59e0b;
}
```

**Tipografia** (Bunny Fonts — GDPR-compliant, não Google Fonts):
```html
<link rel="preconnect" href="https://fonts.bunny.net">
<link href="https://fonts.bunny.net/css?family=inter:400,500,600,700,800|jetbrains-mono:400,500" rel="stylesheet">
```

**Padrão de exportação recomendado:**
```javascript
// styles.js
function getBaseStyles() {
  return `<style>/* tokens + base */</style>
<link rel="stylesheet" href="https://fonts.bunny.net/...">`;
}
module.exports = { getBaseStyles };
```

**Atenção:** Cada página pode ter CSS específico próprio — não remover CSS específico, só remover duplicação dos tokens e estilos base.
