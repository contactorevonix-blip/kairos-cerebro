## Technical Context

**Files:**

- `.aiox-core/version.json` → framework version, installed-at, fileHashes manifest

**Purpose Distinction:**
- `core-config.yaml::project.version` = Kairos Check release version (2.1.0)
- `version.json::version` = AIOX framework version at installation time (5.2.9)
- `version.json::installedAt` = When framework was installed (Jun 2026)
