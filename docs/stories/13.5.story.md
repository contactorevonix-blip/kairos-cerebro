# Story 13.5: Load Squad Configurations

**Story ID:** 13.5  
**Epic:** EPIC-13 (Full Context Determinism)  
**Status:** Ready  
**Points:** 5sp  
**Owner:** @dev  
**Dependencies:** 13.1

## Description

Load all squad configurations from `.claude/rules/squad-*.md` with authority matrix validation. Squads are teams of coordinated agents; each squad must declare its composition, authority, and decision-making rules deterministically.

## Acceptance Criteria

- [ ] Discovers and loads all squad-*.md files from `.claude/rules/`
- [ ] Parses squad structure (name, agents, authority, decision rules)
- [ ] Validates authority matrix (no conflicting permissions)
- [ ] Caches squad configs in `.synapse/squad-cache/{squad-id}.json`
- [ ] Cache TTL: 1 hour (configurable)
- [ ] On squad activation, loads cache if valid, else reloads from disk
- [ ] Detects and reports authority conflicts (BLOCK if any found)
- [ ] Logs cache hit/miss + validation results to `.aiox/context-load-logs/`
- [ ] All 4 pre-commit gates PASS
