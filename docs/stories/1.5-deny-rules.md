# Story 1.5 — Deny Rules Configuration

**Epic:** PHASE 1 — Constitutional Enforcement  
**Story ID:** 1.5  
**Effort:** 5 story points  
**Status:** Draft  

---

## Goal
Configurar DENY rules em `.claude/settings.json` para bloquear operações não autorizadas por Agent Authority.

## Acceptance Criteria

- [ ] Deny rule: `git push` (só @devops pode fazer)
  - Qualquer agente tenta git push → bloqueado com erro claro
  - @devops consegue fazer git push

- [ ] Deny rule: editar `.aiox-core/core/` e `.aiox-core/development/`
  - Qualquer agente tenta editar L1/L2 → bloqueado
  - @architect consegue editar (exceção configurada)

- [ ] Deny rule: `*create-story` sem @sm
  - Qualquer agente tenta criar story → bloqueado
  - @sm consegue criar story

- [ ] Deny rules testadas
  - Tentar git push → mensagem: "Only @devops can push. Use @devops *push" ✅
  - Tentar editar core → mensagem: "Framework core is protected. L1/L2 never modify." ✅
  - Tentar criar story → mensagem: "Story creation is @sm exclusive. Use @sm *create-story" ✅
