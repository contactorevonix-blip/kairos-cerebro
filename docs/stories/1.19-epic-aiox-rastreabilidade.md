### Story 1.19b — Constitution Enforcement Hooks

**Effort:** 8sp | **Owner:** @devops | **Status:** Draft

**Goal:** Implementar pré-commit hooks que bloqueiam AUTOMATICAMENTE qualquer criação sem story ID + validam Art. IV.

**AC:**
- [ ] Hook `.husky/pre-commit` — valida que TODO novo ficheiro (*.md, *.js, *.yaml) tem comentário com `[Story X.Y]` ou `Story-X.Y`
- [ ] Hook bloqueia criação de scripts sem story ID em comentário  
- [ ] Hook bloqueia criação de skills sem story ID em descrição YAML
- [ ] Hook bloqueia criação de templates sem story ID em frontmatter
- [ ] Hook valida Constitution Art. IV: recusa ficheiros que inventam features (heurística: ficheiro novo SEM referência a story = invenção)
- [ ] Hook registra automaticamente em registry: `.aiox-core/data/entity-registry.yaml` com story ID + path
- [ ] Teste: tentar commit sem story ID → BLOQUEADO com mensagem clara

**Output:** 
- Atualizar `.husky/pre-commit` 
- Novo script: `bin/validate-rastreabilidade.js`
- Novo script: `bin/validate-constitution.js`
