## Dev Notes

**Layer classifier** (lib/reader.js):
```javascript
function classifyLayer(relativePath) {
  if (relativePath.startsWith('.aiox-core/core/') ||
      relativePath === '.aiox-core/constitution.md') return 'L1';
  if (relativePath.startsWith('.aiox-core/development/') ||
      relativePath.startsWith('.aiox-core/infrastructure/')) return 'L2';
  if (relativePath.startsWith('.aiox-core/data/') ||
      relativePath === '.claude/settings.json') return 'L3';
  return 'L4';
}
```

**File tree node** response shape (ADR-3):
```json
{ "name": ".aiox-core", "type": "directory", "layer": "L1",
  "layerColor": "#EF4444", "editable": false, "childCount": 8 }
```
