## Dev Notes

**story-parser.js** — extrai de cada MD:
```javascript
function parseStory(content, filename) {
  const statusMatch = content.match(/^## Status\s*\n\*?\*?(\w+)/m);
  const titleMatch = filename.match(/\d+\.\d+\.(.+)\.story\.md$/);
  const executorMatch = content.match(/executor:\s*"(@\w+)"/);
  const hasQaGate = content.includes('qa-gate') && content.includes('PASS');

  return {
    id: filename.split('.')[0] + '.' + filename.split('.')[1],
    title: titleMatch?.[1]?.replace(/-/g, ' ') || filename,
    status: statusMatch?.[1] || 'Unknown',
    executor: executorMatch?.[1] || '@dev',
    hasQaGate,
    sdcProgress: calculateSDCProgress(content),
  };
}
```

**SDC Progress** — baseado em checkboxes `[x]` nas Tasks:
```javascript
function calculateSDCProgress(content) {
  const total = (content.match(/- \[[ x]\]/g) || []).length;
  const done  = (content.match(/- \[x\]/g) || []).length;
  return total > 0 ? Math.round((done / total) * 100) : 0;
}
```
