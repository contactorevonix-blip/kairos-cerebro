## Summary

Implement Agents tab showing status of all 55+ AIOX agents. Display: agent name, status (idle/active), current task, last activity, task count. Update via WebSocket deltas. Card-based layout with status indicators.

**Why:** G1 requires visibility. Agent board shows system load, bottlenecks, idle agents.

---

## Acceptance Criteria

- [ ] Create `ui/src/pages/Agents.tsx`:
  - Grid of agent cards (8-12 per row on desktop, responsive)
  - Each card shows: agent name, status (big, colorful), current task, last activity

- [ ] Card design:
  ```
  ╔══════════════════╗
  ║ @dev             ║
  ║ ● ACTIVE         ║  (● green = active, ○ gray = idle)
  ║ Task: 1.1        ║
  ║ Last: 2s ago     ║
  ║ Total tasks: 42  ║
  ╚══════════════════╝
  ```

- [ ] Status indicators:
  - 🟢 ACTIVE (executing task)
  - 🔵 IDLE (ready, no current task)
  - 🟠 PAUSED (intentionally paused)
  - ⚫ OFFLINE (last activity >5min)

- [ ] Interactivity (read-only):
  - Click card to expand → show task history (last 10 tasks)
  - Expansion read-only (no actions)

- [ ] Sorting:
  - By status (active first)
  - By agent name (alphabetical)
  - By task count (busiest first)

- [ ] Real-time updates:
  - Status badge animates on change
  - Last activity updates every second

- [ ] Tests: card rendering, status display, real-time updates

---

## Implementation Notes

- **Responsive:** CSS Grid auto-scales
- **Color coding:** instant visual identification
- **Read-only:** no activate/stop buttons (Constitution Art. I)
