## Description

Implement Constitutional Articles V-VII enforcement gates: Quality First (Art. V), Code Intelligence (Art. VI-partial), and Framework Boundary (Art. VII). Enhance existing `enforce-quality-gates.cjs` hook.

**Articles V-VII Statements:**
- Art. V: Quality gates (lint, typecheck, test, CodeRabbit) must PASS before merge
- Art. VI: Code intelligence optional but graceful (no failures if unavailable)
- Art. VII: L1/L2 framework core never modified (deny rules in `.claude/settings.json`)
