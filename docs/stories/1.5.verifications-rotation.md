## Story
**As a** founder do Kairos Check,
**I want** que o ficheiro `verifications.jsonl` arquive automaticamente entradas com mais de 90 dias,
**so that** o ficheiro não cresça indefinidamente em produção e as leituras se mantenham rápidas.

## Acceptance Criteria

1. Script `bin/rotate-verifications.js` criado que:
   - Lê `.kairos-data/verifications.jsonl`
   - Move entradas com `timestamp` > 90 dias para `.kairos-data/verifications-archive-YYYY-MM.jsonl`
   - Mantém entradas recentes no ficheiro activo
   - Operação é atómica (tmp+rename — não perde dados se interrompida)
2. O script suporta `--dry-run` (mostra o que faria sem fazer nada)
3. O script suporta `--days N` para configurar o threshold (default: 90)
4. Dados arquivados continuam acessíveis — não são apagados
5. O audit chain (campos `prevHash` e `chainHash`) nas entradas activas permanece válido após rotação
6. Script adicionado ao `package.json` como `"rotate": "node bin/rotate-verifications.js"`
7. `.env.example` ou README documenta como agendar a rotação

**Schema de `verifications.jsonl`** (cada linha é JSON):
```json
{
  "timestamp": "2026-05-24T10:00:00Z",
  "tenantId": "...",
  "decision": "block|review|allow",
  "score": 75,
  "requestId": "uuid",
  "prevHash": "sha256...",
  "chainHash": "sha256...",
  ...
}
```

**Padrão de escrita atómica** (já usado no sniper-db — seguir o mesmo padrão):
```javascript
const tmp = path + '.tmp';
fs.writeFileSync(tmp, content);
fs.renameSync(tmp, path); // atómico no mesmo filesystem
```

**Leitura por stream** (para não carregar GB em memória):
```javascript
const readline = require('readline');
const rl = readline.createInterface({ input: fs.createReadStream(path) });
for await (const line of rl) { /* processar linha a linha */ }
```

**Nota sobre o audit chain:** O chain é entre entradas consecutivas (`chainHash = SHA256(prevHash + body)`). A rotação move entradas antigas para arquivo mas não quebra o chain — as entradas activas continuam a referenciar as suas predecessoras correctas. O arquivo é a prova histórica.

**Threshold em dias:**
```javascript
const DAYS = parseInt(process.argv.find(a => a.startsWith('--days='))?.split('=')[1] || '90');
const cutoff = Date.now() - DAYS * 24 * 60 * 60 * 1000;
```
