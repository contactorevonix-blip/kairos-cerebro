## Story

**As a** framework architect,
**I want** a module that extracts agent communication patterns (tone, vocabulary, emoji style, greeting) into a structured DNA model,
**so that** squad creation can clone the mentor's voice and communication style automatically.

---

## Acceptance Criteria

- [x] **AC1:** Module `squads/squad-creator/core/voice-dna.js` created with `extractVoiceDNA(agentYaml)` function
- [x] **AC2:** DNA model includes: `tone` (string), `signature_vocab` (array of top 10 words), `emoji_style` (frequency + favorites), `greeting_template` (with {placeholders})
- [x] **AC3:** DNA output stored as JSON to `.aiox/squad-dnas/{agent-id}-voice.json` (via `saveVoiceDNA`)
- [x] **AC4:** Unit tests verify extraction accuracy: Clone @dev voice → tone resolves to "direct + pragmatic" (test passing; verified end-to-end against real `dev.md`)
- [x] **AC5:** No invented patterns; all extraction rules map to existing agent YAML structure (persona_profile.communication, persona)

---

## Dependencies

- **Depends on:** None — foundation module for Phase 3
- **Blocks:** 8.3.3 (Squad Template Generation), 8.3.8 (Integration Tests)

---

## Scope

**IN:**
- Agent YAML parsing (persona_profile.communication section)
- Vocabulary frequency analysis from agent markdown descriptions
- Tone classification (direct, empathetic, analytical, etc.)
- Greeting pattern extraction

**OUT:**
- Voice training from live agent behavior (that's Phase 4 auto-healing)
- Feedback loop optimization
- UI/dashboard display (CLI-only for Phase 3)

---

## Risks

- **Risk:** Tone classification too narrow (misses nuance)
- **Mitigation:** Use keyword-based heuristics per agent; document assumptions in code comments
