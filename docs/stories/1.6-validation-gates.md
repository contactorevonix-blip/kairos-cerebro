# Story 1.6 — Validation Gates (Pre-commit/Pre-push)

**Epic:** PHASE 1 — Constitutional Enforcement  
**Story ID:** 1.6  
**Effort:** 3 story points  
**Status:** Draft  

---

## Goal
Implementar automated validation gates que verificam Constitution compliance ANTES de commit/push.

## Acceptance Criteria

- [ ] Pre-commit gate criado
  - Verifica: "A story tem acceptance criteria?" → se não, bloqueia commit
  - Verifica: "Ficheiros modificados são L1/L2?" → se sim, bloqueia (erro claro)
  - Executa sem demora (< 2 segundos)

- [ ] Pre-push gate criado
  - Verifica: "Código tem syntax errors?" → se sim, bloqueia
  - Verifica: "CodeRabbit score >= 7/10?" → se não, warning (não bloqueia)
  - Aviso claro se há issues

- [ ] Gates integrados em `.husky/`
  - Pre-commit hook carrega
  - Pre-push hook carrega
  - Sem erros

- [ ] Testes manuais PASS
  - Tentar commit sem AC → bloqueado ✅
  - Tentar commit com L1/L2 mudança → bloqueado ✅
  - Tentar push com erros → bloqueado ✅
  - Commit/push válidos → funcionam ✅
