# Story 13.15 — Create validate-tool-references.js Utility Validator

**Status:** Done

## Story

**As a** framework contributor,
**I want** a validate-tool-references.js script that validates tool references in the AIOX framework,
**so that** utility integration workflows can automatically verify that referenced agents and tasks exist and are properly configured.

**Epic:** EPIC-13 — AIOX Framework Hardening (Onda 2)

**Story Points:** 5sp
**T-Shirt Size:** S

---

## Acceptance Criteria

- [x] Script `outputs/architecture-map/schemas/validate-tool-references.js` is created and executable
- [x] Script validates that all agent/task references in `core-config.yaml` point to actual files in `.aiox-core/`
- [x] Script generates a structured report with: missing references, invalid paths, validation success/fail count
- [x] Script runs without errors for both valid and invalid configurations
- [x] Script outputs results in JSON format (for tooling) and human-readable format
- [x] Script can be invoked via `node outputs/architecture-map/schemas/validate-tool-references.js`
- [x] Script returns exit code 0 for all valid, exit code 1 for any violations
- [x] No CRITICAL issues from CodeRabbit review
