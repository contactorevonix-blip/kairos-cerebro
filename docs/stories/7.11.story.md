## Summary

Documentar comando oficial de sync em `docs/guides/aiox-playbook.md` (ou novo ficheiro). Pedro pediu "comando para sincronizar tudo" (sessão Cont 25). Upstream documenta **um único comando oficial**:

```bash
npx aiox-core@latest install
```

Este comando é idempotente, detecta instalação existente (v5.2.9 local), actualiza só ficheiros alterados, faz `.bak` de customizações, preserva config do projecto. **Não está documentado localmente.**
