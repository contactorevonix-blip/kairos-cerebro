## Description

Implement validation system that verifies 100% context coverage, detects gaps, and validates atomic layer transitions. This ensures that all 8 SYNAPSE layers load completely and deterministically, with no partial loads or missing rules.

## Acceptance Criteria

- [ ] Verify all 8 layers loaded (Constitution, Global, Agent, Workflow, Task, Squad, Keyword)
- [ ] Count rules loaded and verify >= 500 total rules available
- [ ] Detect missing rules (rules documented but not loaded) — BLOCK if gaps found
- [ ] Validate atomic transitions (no partial layer loads, rollback on error)
- [ ] Generate gap report (`.aiox/validation-reports/coverage-{date}.json`)
- [ ] Update `.synapse/metrics/hook-metrics.json` with `layersLoaded: 8, totalRules: N`
- [ ] Log validation results to `.aiox/context-load-logs/`
- [ ] All 4 pre-commit gates PASS
