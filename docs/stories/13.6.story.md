# Story 13.6: Load Keyword Domain Rules (L6)

**Story ID:** 13.6  
**Epic:** EPIC-13 (Full Context Determinism)  
**Status:** Draft  
**Points:** 4sp  
**Owner:** @dev  
**Dependencies:** 13.1

## Description

Load all L6 keyword domain rules from `.claude/rules/keyword-*.md` on-demand. L6 rules are specialized context loaded only when specific keywords are detected in the active task, enabling dynamic rule loading without bloating agent context.

## Acceptance Criteria

- [ ] Discovers all keyword-*.md files from `.claude/rules/`
- [ ] Parses keyword triggers (when to load each rule)
- [ ] Implements on-demand loading (load only when keywords matched)
- [ ] Caches keyword rules in `.synapse/keyword-cache/{keyword-id}.json`
- [ ] Cache TTL: 1 hour (configurable)
- [ ] On keyword detection, loads cache if valid, else reloads from disk
- [ ] Logs cache hit/miss + keyword triggers to `.aiox/context-load-logs/`
- [ ] All 4 pre-commit gates PASS
