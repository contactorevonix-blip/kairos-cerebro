## Story

## Acceptance Criteria

1. Script `scripts/extract-github.js` executável com `node scripts/extract-github.js`
2. Extrai sem autenticação (repo público MIT) via GitHub API ou raw URLs
3. Extrai os seguintes paths do repo `SynkraAI/aiox-core` branch `main`:
   - `.aiox-core/development/agents/*.md` (12 agentes)
   - `.aiox-core/development/tasks/*.md` (40+ tasks)
   - `.aiox-core/development/workflows/*.md` (4 workflows)
   - `.aiox-core/development/templates/*.yaml` (templates)
   - `.aiox-core/constitution.md`
4. Output guardado em `content/raw/github/` com estrutura de pastas preservada
5. Script imprime log: total de ficheiros extraídos + eventuais erros
6. Script idempotente: re-executar sobrescreve sem duplicar
