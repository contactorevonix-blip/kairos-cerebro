Enable agent memory logging infrastructure for Agent Immortality (v5 feature). Configure `immortality.logging: true` in `core-config.yaml` and create logging framework. Phase 1 (Cont 72) establishes logging; Phase 2-3 (Cont 73+) adds persistence and resurrection.
