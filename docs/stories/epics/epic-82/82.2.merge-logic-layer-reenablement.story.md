## Story

**As the** SYNAPSE engine,
**I want** L3-L7 lazily re-enabled and all active layers merged with deterministic precedence,
**so that** the right context is combined without duplication or constitutional violations.

## Acceptance Criteria

1. Given `active_workflow.id == "story_development"`, when the engine runs, then L3 injects `.synapse/workflow-story-dev` rules.
2. Given no trigger for L4/L5/L6 (no active task, no active squad, no keyword match), when the engine runs, then those layers are skipped (0 rules) without disabling the pipeline or any other layer.
3. Given the same rule text present in both L1 and L2, when merged, then it appears exactly once in the output, attributed to L2 (higher precedence).
4. Given an L7 style rule that conflicts with an L0 constitutional rule, when merged, then the L0 rule is kept and the L7 directive is dropped.
5. Precedence is data-driven: a table loaded from `.synapse/precedence.json` (not hard-coded inside `process()`).
6. Median pipeline time stays below 100ms with L0, L1, L2, L3, and L7 active simultaneously (measured via `engine.js:396` metrics).

### Trigger Evaluation Model (FR-5)

Replace the static `DEFAULT_ACTIVE_LAYERS=[0,1,2]` with:
```js
const activeLayers = [];
activeLayers.push(0, 1);                          // always
if (session.active_agent?.id)    activeLayers.push(2); // L2
if (session.active_workflow)     activeLayers.push(3); // L3
if (session.active_task)         activeLayers.push(4); // L4
if (session.active_squad)        activeLayers.push(5); // L5
if (keywordMatch(prompt))        activeLayers.push(6); // L6
if (/\*\w+/.test(prompt))        activeLayers.push(7); // L7
```
Layers with no trigger produce 0 rules and are skipped cleanly — no hard-disable.

### Merge Algorithm (FR-6/FR-7) — from PRD §3.2

```
results = [ layerResult for each active layer, in layer order L0..L7 ]
PRECEDENCE = loaded from .synapse/precedence.json
merged = []; seen = {}

for r in results sorted by PRECEDENCE[r.layer] desc:
    for rule in r.rules:
        key = normalize(rule)
        if key in seen: continue          // dedup — keep higher precedence
        if violatesConstitution(rule): drop  // L0 never overridden
        seen[key] = r.layer
        merged.push({ layer: r.layer, rule })

// Render in layer order (L0 section first); token budget trims lowest-precedence first (NFR-2)
```

Constitution-wins: `violatesConstitution(rule)` — any rule that directly contradicts an L0 rule is dropped, not just deduped.
