# KAIROS Check — Privacy Policy

Last updated: 2026-05-27

## 1. Data We Collect

KAIROS Check collects only the data necessary to perform fraud scoring:
- Email addresses submitted for verification
- IP addresses (for rate limiting)
- API usage metrics (anonymous)

## 2. How We Use Your Data

Data submitted to the KAIROS API is used exclusively to return a fraud risk score.
We do not store, sell, or share personal data with third parties.

## 3. Data Retention

Verification requests are not stored beyond the immediate API response.
Aggregate metrics are retained for 90 days for service improvement.

## 4. GDPR Rights

If you are in the EU, you have the right to access, rectify, or delete any data
we hold about you. Contact: privacy@kairoscheck.net

## 5. Contact

For privacy enquiries: privacy@kairoscheck.net
