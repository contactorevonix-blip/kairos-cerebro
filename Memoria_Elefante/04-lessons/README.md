# Lições Estratégicas

> Post-mortems **CEO-level** — não bugs nem incidents técnicos (esses vão para `.ai/`). Aqui ficam decisões de produto, mercado, pricing, hiring e narrativa que correram bem ou correram mal.

## Princípio

Cada lição responde a 3 perguntas:
1. **O que aconteceu** (factos, sem interpretação)
2. **O que eu acreditava antes** (modelo mental anterior)
3. **O que passo a acreditar agora** (modelo mental novo + regra accionável)
